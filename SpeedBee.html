Help

Hello there, simply hit the ctrl key to start the timer and focus on the input to start typing. Hit the ctrl key when you're done to stop the timer and see your results. Beware it is case sensitive and so everything must be typed exactly as the one in the excerpt.

There are seven levels: Noob, Novice, Pro, Expert, Master, Grand Master and Legend. Try and beat these levels and get to the Legendary level.

Have fun!

About

SpeedBee is a program meant to test your typing speed and evaluate how long it takes you to type the given text. SpeedBee is recommended for physical keyboard input
